API para un blog

Esta API fue desarrollada en el entorno de Nodejs con el framework Expressjs, esta API maneja las operaciones básicas de un blog, es decir, agregar artículos, editar artículos, eliminar artículos y obtener artículos ya sea uno en específico o listarlos a todos. Los artículos son guardados en una base de datos de MongoDB, para facilitar estas operaciones usé Mongoose. Ver código fuente

Gestor de horarios

Este sistema permite manejar horarios de instituciones educativas tomando en cuenta aulas, profesores y grupos asignados. Básicamente permite que los profesores inicien sesión (las cuentas deben ser previamente creadas por el administrador del sistema) y una vez dentro del sistema pueden elegir el horario y el aula que prefieran, el sistema verifica que no hayan conflictos y realiza la asignación. La API está desarrollada en .NETCore y utiliza una base de datos de SqlServer. La parte del frontend está desarrollada en React. Ver backendVer fronted
